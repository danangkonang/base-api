# simple crud go


## installation

```bash
git clone https://github.com/danangkonang/simple-crud-golang.git crud-rest

cd crud-rest
```

## database

- this project use tool _gomig_ for migration. you can red doc in [here](https://github.com/danangkonang/migration-go-cli)

```bash
./gomig run migration
```

## running program
```go
go run main.go
```
